Lab 9: Node.js & Express Server

By Claudia Herrera

Overview

This lab introduced the use of Node.js with the Express framework to create a simple server that serves both static and dynamic content. We used fetch() to pull data from JSONPlaceholder and practiced chaining promises using .then() and .catch().

Technologies Used

  • Node.js
  • Express.js
  • JavaScript (ES6)
  • HTML
  • JSONPlaceholder API

What I Learned

I learned how to set up a basic Express server, serve static files from a public folder, and create dynamic API routes using /photos and /photos/:id. This helped me better understand backend routing and asynchronous data fetching with JavaScript.

Demo

To see the server in action, visit:

  • localhost:8080/photos – View the first 20 photos
  • localhost:8080/photos/3 – View photo with ID 3
