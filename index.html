Lab 9: Node.js & Express Server

By Claudia Herrera

Overview

This lab introduced the use of Node.js with the Express framework to build a basic server that serves static and dynamic content. It demonstrates how to use the fetch() API to retrieve external data and handle asynchronous operations with JavaScript promises.

Technologies Used

  • Node.js
  • Express.js
  • JavaScript ES6
  • JSONPlaceholder API
  • Git & GitHub

What I Learned

I practiced backend development by building routes, serving static files, and handling asynchronous data fetching with promise chains. I also learned how to use nodemon for development and structure a project for deployment on GitHub Pages.

Demo Routes

  • /photos/3 – Single photo object by ID
